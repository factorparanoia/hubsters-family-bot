# HUB Bot — повнофункціональна заміна ProBot/CarlBot

Оновлено: переписано критичні системи музики, тікетів та логів/активності.

## ✅ Що виправлено

### 1) Музика (voice playback)
- Бот стабільно підключається до voice (перевірка прав Connect/Speak).
- Якщо підключення неможливе — повертає точну причину.
- Відтворення працює через `@discordjs/voice` + `play-dl`.
- Бот у voice працює з `selfDeaf: true` (не слухає людей).

### 2) Тікети з кнопками
- Нова команда `/ticket_panel` публікує кнопку **"Открыть"**.
- При натисканні створюється приватний тікет з нумерацією: `ticket-0001`, `ticket-0002`, ...
- У тікеті є службовий embed:
  - хто створив,
  - номер тікета,
  - умови/правила тікета,
  - кнопка **"Закрыть"**.
- Закрити тікет може автор або модератор.

### 3) Детальні логи та активність
- Збирається статистика по кожній людині:
  - кількість повідомлень,
  - вкладення,
  - використані команди,
  - активність по каналах,
  - голосова активність (секунди, входи).
- Збирається статистика по каналах:
  - кількість повідомлень,
  - топ активних користувачів.
- Додані команди:
  - `/activity_user`
  - `/activity_channel`

### 4) Більш детальна інформація
- `/userinfo` тепер показує детальну активність користувача.
- `/channelinfo` показує статистику каналу і топ активних людей.
- `/serverinfo` показує розширений профіль сервера.

## Railway
- Налаштуйте `.env`: `DISCORD_TOKEN`, `DISCORD_CLIENT_ID`, `DISCORD_GUILD_ID`
- Запуск: `npm start`
- Slash-команди синхронізуються автоматично.
